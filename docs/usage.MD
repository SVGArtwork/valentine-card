# Guida all’uso

## Avvio
- Aprire `index.html` nel browser.
- L’animazione parte automaticamente:
  1. Stato `LOADING`
  2. Intro (titoli iniziali)
  3. Animazione principale tappo e bollicine
  4. Stato `IDLE` con animazioni lampadine e bollicine

## Interazioni
- **Pannello info**: cliccare sugli elementi SVG definiti (`Gatto`) per vedere titolo e descrizione.
- **Zoom e Pan**:
  - Slider o pulsanti `zoom-in` / `zoom-out`.
  - Drag per spostare SVG.
- **Restart animazione**: cliccare il bottone con ID `restart`.
- Tutti i parametri animazione sono configurabili via JS.

## Note
- La funzione globale `avviaAnimazionePrincipale()` permette di riavviare manualmente l’animazione del tappo.
