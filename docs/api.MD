# API Funzioni

Queste funzioni sono accessibili globalmente:

- **`avviaAnimazionePrincipale()`**
  - Riavvia l’animazione del tappo se non già in corso.

- **`setState(newState)`**
  - Cambia lo stato della state machine.
  - Stati disponibili: `LOADING`, `INTRO`, `ANIMATION`, `IDLE`.

- **Elementi SVG con ID**
  - `Sughero`, `Traiettoria`, `Skizzo_1`, `Skizzo_2`, `Schiuma_Sopra`, `Schiuma_Sotto`, `Mano_Flut`, `Smile`, `Occhi_Gatto_01`, `Testo`
  - Permettono interazioni e animazioni controllate via JS.
